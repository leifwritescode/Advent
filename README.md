# Advent of Code 2019
Solutions in Kotlin

## Usage
```
make 
```
Will build the solutions for 2019.
```
make run D=<0..25>
```
Where `D` is any non-zero integer up to 25, will run the solution for DD/12/2019.
```
make clean
```
Will remove all build artefacts
