# Advent of Code 2019
Solutions in Kotlin

## Usage
Where `day` is a non-zero integer between 1 and 25 inclusive:
```
make DAY=<day>
```
Will build and execute the solution for `day`.

To remove **all** build artefacts:
```
make clean
```
